## 准备东西

[MCL一键启动器](https://github.com/iTXTech/mcl-installer/releases)

Archlinux用户可以直接在aur安装，默认是安装到`/opt/mcl`里面的，路径权限给的`777`(汗～)

这个东西可以让你直接在服务器上登陆上QQ，就和MC启动器一样

但是，里面是干干净净的，啥玩意儿都没有！
